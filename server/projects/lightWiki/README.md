Wiki leggera costruita in PHP + SQLite, con supporto Markdown esteso, diagrammi Mermaid e integrazione AI per ricerca semantica.

Sviluppata per la **NASA Space Apps Challenge** (ottobre 2025) - team **più giovane d'Italia** dell'edizione.

## Funzionalità

- **Markdown esteso** - tabelle, footnote, syntax highlighting out-of-the-box
- **Mermaid.js** - diagrammi e flowchart embedded nelle pagine
- **SQLite backend** - nessun database esterno da configurare
- **REST API** - crea, modifica e cancella pagine da script esterni
- **Storico revisioni** - ogni edit è tracciato con diff
- **Full-text search** - ricerca su tutto il contenuto
- **Autenticazione e ruoli** - gestione utenti integrata
- **AI semantica** - ricerca per significato tramite embedding (attualmente disabilitata per motivi logistici, infrastruttura pronta)
- **Plugin e theme system** - estendibile senza modificare il core

**Prototipo live:** [bsl.mywire.org](http://bsl.mywire.org)

## Stack

PHP · SQLite · JavaScript · CSS · embedding vectors · Composer
