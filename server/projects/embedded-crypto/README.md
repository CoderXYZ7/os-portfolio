Sistema di **criptaggio asimmetrico** per dispositivi embedded sviluppato durante il quinto anno delle superiori per una fondazione (nome riservato per accordo di riservatezza).

Il dispositivo opera **senza accesso diretto a internet**: il traffico viene instradato tramite un device intermedio con accesso controllato, creando un layer di isolamento fisico e logico.

Il progetto è coperto da proprietà intellettuale ed è attualmente **in produzione**. Non è possibile fornire ulteriori dettagli tecnici.

**Caratteristiche pubbliche:**
- Criptografia asimmetrica su hardware embedded
- Architettura air-gap parziale con proxy controllato
- Deployment in ambiente produttivo reale

**Stack:** C++ · embedded Linux · crittografia asimmetrica · networking controllato
